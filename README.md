# SuperKart Sales Forecast Space

Streamlit demo that predicts Product_Store_Sales_Total for SuperKart dataset. Enter a single record or upload a CSV for batch predictions.

- Model: `aldon-cabral/SuperKart-RF`
- Space: `aldon-cabral/SuperKart-space`

Make sure `requirements.txt` includes runtime dependencies and add any secrets (HF_TOKEN, HF_MODEL) in the Space settings if the model is private.
